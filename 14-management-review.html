管理评审

管理评审是最高管理者对质量管理体系的系统性评价，也是注册质量经理考试的高频考点。核心在于掌握输入和输出内容。

输入
Input
评审依据和信息
输出
Output
评审结论和决策
频次
Frequency
按计划的时间间隔
主持
Host
最高管理者主持
15.1.1

管理评审输入资料

1. 内审结果

内部审核的发现和问题

2. 客户反馈

客户满意度、投诉、建议等

3. 供方绩效

供应商质量、交付、服务表现

4. 上次评审跟踪

上次管理评审整改措施的跟踪情况

5. 过程绩效

各过程的绩效数据和趋势

6. 产品符合性

产品质量目标的达成情况

⚠️ 考试高频陷阱：管理评审输入不包含"产品出厂检验报告"。产品出厂检验报告是日常质量记录，不是管理评审的输入资料。
15.1.2

管理评审输出

1. 体系改进决议

对质量管理体系的改进决策

2. 资源调整需求

人力、设备、资金等资源的调整

3. 质量目标更新

质量目标的修订和更新

❌ 不包含

产品出厂检验报告

管理评审输出三要素：体系改进决议 + 资源调整需求 + 质量目标更新。
15.1.3

关键要点

主持人

最高管理者亲自主持，不能委托他人

频次

按计划的时间间隔进行，通常每年至少一次

目的

评价体系的适宜性、充分性和有效性

记录

应保留管理评审记录，包括输入、输出和决策
